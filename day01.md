## 💻 Cloud Computing 
Cloud computing is the on-demand availability of computing resources (Eg, Server, Storages etc) over the internet connectivity. (Example, AWS, Azure, Google Cloud etc). 
* EC2 (Elastic Compute): EC2 is basically a virtual server in the cloud. 


## VPC
Virtual Private Cloud (VPC) is a secure private network/infrastructure deployed in the AWS public cloud. It lets you launch your AWS resources into a virtual network that you define and control. Intially **Default VPC** has decleared but for security we'll go for **Custom VPC**. 

### The main elements of a VPC:
* Subnets
* Routing Table
* Internet Gateway
* NAT
